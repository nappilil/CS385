## How To Run Project

- For the backend:
    ```bash
    cd server
    ```

1. Install packages
    ```bash
    npm i @apollo/server
    npm i flat
    npm i graphql
    npm i mongodb
    npm i redis
    npm i seed
    ```

2. Run the seed file
    ```bash
    npm run seed
    ```

3. Start the server
    ```bash
    npm start
    ```

- For the frontend:
    ```bash
    cd react_client
    ```
1. Install packages
    ```bash
    npm i @apollo/client
    npm i graphql
    npm i react
    npm i react-dom
    npm i react-modal
    npm i react-router-dom
    ```
2. Start the server
    ```bash
    npm run dev
    ```
